# Desktop Configs

Configs I use for my Debian/i3 desktop setup.

This repo is mainly for personal backup and for quickly restoring parts of my environment on another machine.

## Included configs

- `i3/` - i3 window manager config
- `i3status/` - i3status bar config
- `nvim/` - Neovim config
- `picom/` - compositor config
- `rofi/` - app launcher config
- `gtk-3.0/` - GTK 3 appearance settings
- `gtk-4.0/` - GTK 4 appearance settings
- `fastfetch/` - terminal system info config
